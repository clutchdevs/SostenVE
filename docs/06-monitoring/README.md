## Qué irá aquí
- Seguimiento de las **métricas de éxito** del charter:
  - Tiempo de asignación (mediana, por nivel de riesgo).
  - Cobertura de riesgo alto.
  - Tasa de abandono de cola.
  - Disponibilidad del sistema (uptime).
- Monitoreo de capacidad y cola en tiempo real (panel del coordinador).
- Alertas técnicas (errores, caídas, fallos de respaldo).
- Verificación periódica de que los respaldos se completan correctamente.
